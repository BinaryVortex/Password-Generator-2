# Password Generator 2

A simple, web-based password generator built with HTML, CSS, and JavaScript.

## Features

- Generate secure passwords with customizable length (4-32 characters)
- Include/exclude uppercase letters, lowercase letters, numbers, and symbols
- One-click copy to clipboard
- Responsive design

## Screenshot

![Password Generator Screenshot](Screenshot%202024-06-14%20143732.png)

## How to Use

1. Open `index.html` in your web browser.
2. Adjust the password length using the slider.
3. Check/uncheck the options for character types.
4. Click "Generate Password" to create a new password.
5. Click the copy icon to copy the password to your clipboard.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
